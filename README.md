# ⚡ 電力爸爸的互動遊戲庫

> 用可愛的海洋生物，帶孩子認識電力的世界 🌊

歡迎來到「電力爸爸的生活筆記」互動遊戲庫！

我是電力爸爸，一位在台電工作的工程師爸爸。
我把每天工作接觸到的電力知識，變成一個個小遊戲和圖文故事，
希望讓國小的孩子（還有陪讀的爸爸媽媽）能在玩樂中，
輕鬆看懂電是怎麼發、怎麼用、怎麼調度的。

在這裡，電力概念都由海洋電力學校的角色們擔綱演出——
沉穩的電鱝老師、愛計算的海龜助教、負責警示的河豚，
還有代表各種發電方式的能源夥伴們。

## 🌊 遊戲主題

- ⚡ 供電基礎——備轉容量、供電燈號、一度電是多少
- ☀️ 太陽能——鴨子曲線、淨尖峰能力
- 🔋 儲能與調度——抽蓄水力、電網怎麼平衡
- 📊 尖峰離峰——為什麼晚上 6 點用電最多
- 還有更多陸續登場！
